How Stacks Work
A stack is a linear data structure that follows the LIFO (Last In, First Out) principle. Think of a stack of plates — you can only add or remove from the top.
1
Push: Add an element to the top. O(1) — just place it on top.
2
Pop: Remove and return the top element. O(1) — the last added is the first removed.
3
Peek: View the top element without removing it. O(1) — useful to check what's next.
4
Overflow / Underflow: Pushing to a full stack = overflow. Popping from empty = underflow. Always check first.
