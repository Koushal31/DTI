How Linked Lists Work
A linked list is a sequence of nodes. Each node stores a value and a pointer to the next node. Unlike arrays, nodes are not stored in contiguous memory — they can be anywhere, connected by pointers.
1
Node structure: Each node has a data field and a next pointer to the following node.
2
Head pointer: The list is accessed via a head pointer. The last node's next is null.
3
Traversal: Start at head and follow next pointers until you reach null.
4
Insert at head: Create new node, set its next to current head, update head. O(1).
5
Delete a node: Find the predecessor and redirect its next to skip the deleted node.
