How Graph Traversal Works
Both BFS and DFS systematically visit every node in a graph. The difference is the order of visiting — BFS explores all neighbours first (level by level) while DFS dives as deep as possible before backtracking.
1
BFS uses a Queue: Enqueue the start node. Dequeue it, mark as visited, enqueue all unvisited neighbours. Repeat until the queue is empty.
2
DFS uses a Stack (or recursion): Push the start node. Pop it, mark as visited, push all unvisited neighbours. Dive deep before going wide.
3
Visited set: Both algorithms keep track of visited nodes to avoid revisiting them in graphs with cycles.
